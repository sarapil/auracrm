# دليل البدء السريع — AuraCRM

## مرحباً بك في AuraCRM!

AuraCRM هو نظام إدارة علاقات العملاء الشامل المبني أصلاً على Frappe/ERPNext. سيساعدك هذا الدليل على البدء في 10 دقائق.

---

## الخطوة 1: الوصول إلى AuraCRM

بعد التثبيت، انتقل إلى رابط موقعك وسجّل الدخول. يظهر AuraCRM في الشريط الجانبي تحت مساحة عمل **AuraCRM**.

## الخطوة 2: تهيئة الإعدادات

انتقل إلى **إعدادات AuraCRM** (شريط البحث → "AuraCRM Settings"):

1. **التبويب العام**: حدد شركتك والعملة الافتراضية
2. **تبويب التقييم**: اختر نموذج التقييم الافتراضي
3. **تبويب اتفاقية الخدمة**: حدد أوقات الاستجابة الافتراضية
4. **تبويب الألعاب**: فعّل إذا كنت تريد لوحات صدارة الوكلاء
5. **تبويب الترخيص**: أدخل مفتاح الترخيص للميزات المميزة (اختياري)

## الخطوة 3: إنشاء أول عميل محتمل

1. انتقل إلى **Aura Lead** ← جديد
2. املأ: اسم العميل، البريد الإلكتروني، الهاتف، المصدر، الصناعة
3. احفظ — سيحسب النظام تقييم العميل تلقائياً
4. يظهر العميل على لوحة خط الأنابيب

## الخطوة 4: إعداد خط الأنابيب

1. انتقل إلى صفحة **لوحة خط الأنابيب**
2. يتم تنظيم العملاء حسب المرحلة: جديد ← تم الاتصال ← مؤهل ← عرض ← تفاوض ← فاز/خسر
3. اسحب وأفلت العملاء بين المراحل
4. انقر على بطاقة العميل للعرض السريع

## الخطوة 5: تعيين العملاء للوكلاء

### التعيين اليدوي
- افتح عميل محتمل ← حدد حقل **مالك العميل**

### التوزيع التلقائي
1. أنشئ **قاعدة توزيع العملاء المحتملين**
2. اختر الطريقة: دوري، حسب السعة، أو مطابقة المهارات
3. أضف الوكلاء إلى القاعدة
4. سيتم تعيين العملاء الجدد تلقائياً

## الخطوة 6: تتبع التفاعلات

في أي عميل محتمل، انقر **+ تفاعل** لتسجيل:
- 📞 مكالمات هاتفية (المدة، النتيجة)
- 📧 رسائل بريد إلكتروني (مرتبطة ببريد Frappe)
- 🤝 اجتماعات (ملاحظات، الخطوات التالية)
- 📝 ملاحظات (تعليقات داخلية)
- 💬 واتساب (إذا تم تهيئته)

## الخطوة 7: تحويل العملاء المحتملين

عندما يكون العميل جاهزاً:
1. افتح العميل ← انقر **تحويل إلى فرصة**
2. املأ قيمة الصفقة وتاريخ الإغلاق المتوقع
3. تابع التتبع في خط أنابيب الفرص

## الخطوة 8: مراقبة الأداء

- **لوحة الفريق**: نظرة عامة على نشاط جميع الوكلاء
- **لوحة خط الأنابيب**: إدارة مرئية لخط الأنابيب
- **التقارير**: قمع التحويل، سرعة خط الأنابيب، أداء الوكلاء

---

## ماذا بعد؟

- **إعداد قواعد التقييم**: AuraCRM > قاعدة تقييم العملاء المحتملين
- **تهيئة SLA**: AuraCRM > سياسة اتفاقية مستوى الخدمة
- **تفعيل الألعاب**: إعدادات AuraCRM > تبويب الألعاب
- **استكشف ميزات الذكاء الاصطناعي** (مميز): تقييم ذكي، إنشاء محتوى، OSINT
- **اقرأ دليل المستخدم الكامل**: docs/USER_GUIDE.md

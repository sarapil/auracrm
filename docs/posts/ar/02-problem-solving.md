## يعمل مع ERPNext الحالي لديك

AuraCRM لا يستبدل ERPNext — بل **يُوسّعه**. ثبّته بجانب إعدادك الحالي:

```bash
bench get-app https://github.com/sarapil/auracrm
bench --site your-site install-app auracrm
```

كل بياناتك وسير عملك وتخصيصاتك الحالية تبقى كما هي.

# AuraCRM — Training Curriculum

## Overview

Structured training program for AuraCRM users, organized by role and skill level.

---

## Track 1: Sales Agent (8 hours)

### Module 1.1: Getting Started (1 hr)
- Logging in and navigating the workspace
- Understanding your dashboard
- Setting up your profile and preferences
- **Lab**: Navigate to Pipeline Board, view assigned leads

### Module 1.2: Lead Management (2 hrs)
- Creating and editing leads
- Understanding lead fields and stages
- Lead scoring — what the score means
- Interaction logging (calls, emails, notes)
- **Lab**: Create 5 leads, log interactions, update stages

### Module 1.3: Pipeline Workflow (2 hrs)
- Pipeline Board — drag-and-drop stages
- Converting leads to opportunities
- Deal tracking and forecasting
- SLA awareness — response time expectations
- **Lab**: Move leads through pipeline, convert one to opportunity

### Module 1.4: Communication (1.5 hrs)
- Email templates
- WhatsApp messaging
- Call logging
- Meeting scheduling
- **Lab**: Send email from template, log a call, create a meeting

### Module 1.5: Reporting (1.5 hrs)
- Personal performance metrics
- Activity reports
- Gamification — understanding points and leaderboards
- **Lab**: View your scores, check leaderboard position

---

## Track 2: Sales Manager (6 hours)

### Module 2.1: Team Dashboard (1.5 hrs)
- Team performance overview
- Agent activity monitoring
- Pipeline health metrics
- SLA breach tracking
- **Lab**: Identify underperforming agents, analyze pipeline bottlenecks

### Module 2.2: Lead Distribution (1.5 hrs)
- Round-robin configuration
- Capacity-based routing
- Skills matching
- Manual reassignment
- **Lab**: Configure distribution rules, test with sample leads

### Module 2.3: Campaigns & Marketing (1.5 hrs)
- Marketing lists
- Campaign creation and execution
- Nurture sequences
- Campaign analytics
- **Lab**: Create a campaign, add leads, review results

### Module 2.4: Advanced Reporting (1.5 hrs)
- Script reports (Conversion Funnel, Pipeline Velocity, Agent Performance)
- Custom filters and date ranges
- Exporting data for presentations
- **Lab**: Generate all 3 script reports, create a weekly summary

---

## Track 3: CRM Administrator (10 hours)

### Module 3.1: AuraCRM Settings (2 hrs)
- All settings tabs walkthrough
- API key configuration
- License management
- Feature flags overview
- **Lab**: Configure all settings sections

### Module 3.2: Roles & Permissions (2 hrs)
- 5 CRM roles explained
- Custom role creation
- Row-level security
- Permission debugging
- **Lab**: Create a restricted role, test access

### Module 3.3: Automation Setup (2 hrs)
- Scheduler events configuration
- Nurture sequence builder
- Auto-assignment rules
- Holiday guard configuration
- **Lab**: Set up a 3-step nurture sequence with holiday guard

### Module 3.4: Intelligence Configuration (2 hrs)
- AI engine setup (API keys, models)
- OSINT engine sources
- Enrichment pipeline
- Content generation settings
- **Lab**: Configure AI scoring, run OSINT on test company

### Module 3.5: Integration Management (2 hrs)
- WhatsApp Cloud API setup
- Email configuration
- ERPNext sync settings
- Social media connections
- **Lab**: Set up WhatsApp integration, test message flow

---

## Track 4: Developer (8 hours)

### Module 4.1: Architecture (2 hrs)
- App structure overview
- Engine pattern understanding
- DocType relationships
- API endpoint catalog
- **Lab**: Read COOKBOOK.md, trace a lead through the code

### Module 4.2: Customization (2 hrs)
- Custom fields on CRM DocTypes
- Client scripts
- Server scripts
- Custom reports
- **Lab**: Add a custom field, create a client script

### Module 4.3: Extension Development (2 hrs)
- Building AuraCRM extensions
- Hook points available
- Custom engine creation
- Feature flag integration
- **Lab**: Build a simple scoring extension (see EXTENSION_GUIDE.md)

### Module 4.4: Testing & Deployment (2 hrs)
- Running test suite
- Writing new tests
- CI/CD pipeline
- Production deployment checklist
- **Lab**: Run tests, add a test for a custom feature, trigger CI

---

## Assessment

### Sales Agent Certification
- [ ] Create and manage 10 leads end-to-end
- [ ] Use pipeline board effectively
- [ ] Log 5 different interaction types
- [ ] Achieve 80% on written quiz

### Sales Manager Certification
- [ ] Configure distribution rules
- [ ] Create and analyze a campaign
- [ ] Generate all 3 script reports
- [ ] Achieve 80% on written quiz

### Admin Certification
- [ ] Complete full settings configuration
- [ ] Set up 2 integrations
- [ ] Configure automation rules
- [ ] Achieve 80% on written quiz
